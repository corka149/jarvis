# jarvis-cli

```
This small CLI helps to manage the jARVIS platform.
It focuses those parts that are not available via WebUI.

Usage:
  jarvis-cli [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  help        Help about any command
  orga        Management of organisations
  user        User management

Flags:
  -h, --help   help for jarvis-cli

Use "jarvis-cli [command] --help" for more information about a command.
```
